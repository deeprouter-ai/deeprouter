
Verified:

- Opted-out users receive no send and no `skill_notification_sent` analytics.
- Saved/downloaded `one_time` promo inputs send re-engagement only to opted-in users.
- Lapsed enabled Skills send "continue using" re-engagement only to opted-in users.
- Open/click tracking accepts only `digest`/`reengage` and persists `skill_notification_opened` / `skill_notification_clicked`.
