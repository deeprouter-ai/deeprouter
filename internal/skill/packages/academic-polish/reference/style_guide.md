# Academic Writing Style Guide

Reference for the Academic Paper Polish skill. Use this to understand what
the AI editor checks and to manually review its suggestions.

---

## 1. Verb Tense Conventions

| Section | Tense | Example |
|---------|-------|---------|
| Abstract | Past (your work) + Present (implications) | "We developed X. X enables Y." |
| Introduction — background | Present | "Transformer models achieve..." |
| Introduction — gap | Present | "No study has examined..." |
| Methods | Past | "We collected... We applied..." |
| Results | Past | "Accuracy improved by 12%." |
| Discussion | Mixed | "Our results suggest (present) that X occurred (past) because..." |
| Conclusion | Past + Present | "This study demonstrated... Future work should..." |

---

## 2. Common Weak Phrases → Stronger Alternatives

| Avoid | Use instead |
|-------|-------------|
| "It is important to note that" | Delete or integrate into claim |
| "In order to" | "To" |
| "Due to the fact that" | "Because" |
| "A large number of" | "Many" / specify count |
| "It can be seen that" | Delete; state the observation directly |
| "We believe that" | "We argue that" / "Evidence suggests that" |
| "Very", "quite", "really" | Delete or quantify |
| "Utilize" | "Use" |
| "Facilitate" | "Enable" / "Help" |
| "Methodology" | "Methods" (unless discussing meta-research) |

---

## 3. Paragraph Structure

Each paragraph should follow **PEEL**:
- **P**oint — topic sentence stating one clear claim
- **E**vidence — data, citation, or example supporting the claim
- **E**xplain — analysis connecting evidence to claim
- **L**ink — transition to next point

Red flags to fix:
- Paragraph with more than 8 sentences (split it)
- Paragraph without a citation in Methods/Results (add or justify)
- Two consecutive paragraphs starting with the same word

---

## 4. Hedging Language (Calibrated Confidence)

Use hedging to match your evidence strength. Overconfident claims invite
rejection; over-hedged writing looks weak.

| Strength | Hedges |
|----------|--------|
| Strong (direct evidence) | "demonstrates", "confirms", "shows" |
| Moderate (consistent pattern) | "suggests", "indicates", "supports" |
| Weak (limited data) | "may", "could", "appears to", "is consistent with" |
| Speculative | "we hypothesize", "it is possible that" |

---

## 5. Citation Conventions

- Cite claims that are not common knowledge
- Cite data/statistics always, even if widely quoted
- Do NOT cite methodology steps that are standard practice
- Format: follow journal style (APA / IEEE / Vancouver)
  - The polish script preserves citation numbers/keys unchanged

---

## 6. Abstract Checklist

A strong abstract has:
- [ ] Problem statement (1–2 sentences)
- [ ] Gap in existing knowledge (1 sentence)
- [ ] Your approach / what you did (1–2 sentences)
- [ ] Key result (1–2 sentences with a number if possible)
- [ ] Implication / significance (1 sentence)
- [ ] Total length: 150–250 words (check journal limits)
- [ ] No citations, no abbreviations not defined in abstract, no figures

---

## 7. Transition Words by Function

| Function | Words / Phrases |
|----------|-----------------|
| Add information | Furthermore, Moreover, In addition, Additionally |
| Contrast | However, Nevertheless, In contrast, Conversely |
| Cause/Effect | Therefore, Consequently, Thus, As a result |
| Illustrate | For example, For instance, Specifically, Notably |
| Sequence | First, Subsequently, Finally, Prior to, Following |
| Concede | Although, While, Despite, Granted that |
| Summarize | In summary, Overall, Taken together |

---

## 8. Numbers and Statistics

- Write out numbers one through nine; use digits for 10+
- Always include units: "3.2 ms", not "3.2"
- Report means ± SD or ± SE (be consistent)
- p-values: report exact values (p = 0.023), not just p < 0.05
- Percentages: "12%" not "12 percent" in results; "12 percent" in discursive text
- Significant figures: match precision to measurement accuracy

---

## 9. Passive vs. Active Voice

**Active** is preferred in modern journals for methods and findings:
- ✓ "We collected 1,200 samples over three months."
- ✗ "1,200 samples were collected over three months."

**Passive** is acceptable when:
- The actor is unknown or irrelevant
- Following field conventions (some biology journals prefer passive)

---

## 10. Common Non-Native Speaker Patterns to Fix

| Pattern | Fix |
|---------|-----|
| "the literatures" | "the literature" (uncountable) |
| "researches" | "research" (uncountable) / "studies" |
| "informations" | "information" (uncountable) |
| "feedbacks" | "feedback" (uncountable) |
| Missing article: "accuracy improved" | "The accuracy improved" / "Accuracy improved" (context-dependent) |
| "propose to" + verb | "propose" + gerund or noun |
| "compare to" (similarity) vs "compare with" (difference analysis) | Use "compared with" in methods |
